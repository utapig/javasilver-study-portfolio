## 3. 要件サマリー

### 3.1 機能要件
- 機能1: 問題を1問ずつ表示し、4択から回答できる
- 機能2: 回答後に正誤と解説を表示し、クリックで次の問題へ進める
- 機能3: 全問終了後に結果ランクを表示し、リトライできる

### 3.2 非機能要件
- JSON 生成時に `"`、`\\`、改行をエスケープし不正JSONを防ぐ
- JavaScript 側で取得失敗時と問題未登録時のメッセージを表示する
- 学習用として可読性を優先し、クラス責務を Main / Question / QuizExporter に分離する
